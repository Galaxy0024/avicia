Chestcount Mod



To use the Chestcount Mod you need to install the ChatTriggers mod for Minecraft version 1.12.2 (Legacy 0.18.4):


When you have installed ChatTriggers you can download the Chestcount Mod

Every time you open a chest, the mod displays the amount of chests you have opened this session and your total chestcount.



To install the Chestcount Mod simply extract ChestcountMod.zip into .minecraft/config/ChatTriggers/modules/

If you don't have that folder, start your game once with ChatTriggers installed and if you still don't have it you can try manually creating the folder.



You will get this error message every time you start your game

It's not important and can be disabled

To disable the error message go to .minecraft\config\ChatTriggers.json and change the value of showUpdatesInChat to false
